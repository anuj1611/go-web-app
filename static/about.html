About Me

Hello! I'm Anuj Dhiraj Bhagat — a final-year engineering student passionate about DevOps, cloud computing, and machine learning.

My technical journey includes building full-stack DevOps pipelines, integrating cloud-native apps with Kubernetes, ArgoCD, GitHub Actions, and AWS services. I recently completed a comprehensive DevOps project based on a Go web app, involving CI/CD, containerization, and infrastructure automation.

I’ve also worked on ML-based optimization projects using Support Vector Machines (SVM), Random Forests, and Genetic Algorithms to design antennas with precise output characteristics.

Along the way, I’ve gained hands-on experience with tools like Docker, Jenkins, Terraform, MySQL, Oracle DB, GCP, and Salesforce, and completed certifications like Cisco Networking Basics and AWS Cloud Practitioner prep.

Outside of academics, I enjoy learning through open-source tutorials and believe in sharing knowledge to grow together. I’m always looking for opportunities to contribute, collaborate, and explore real-world tech challenges.
